Nice animation Progress Bar is an awesome, super animation plugins for your wordpress website. 

== Description ==

This plugin will enable animation Progress bar in your wordpress theme. You can embed  Progress bar via shortcode in everywhere you want, even in theme files.
http://codertanvir.com/projects/btp_hair/animation-progressbar/

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want news ticker in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates

Shortcodes

*. Genarel animation Progressbar

<pre>[progressbar_list id="progressbar_one"  value="10"]</pre>

*.Progress Bar color

<pre>[progressbar_list id="progressbar_2" color="#00A387" value="20" ]</pre>

*.Progress Bar width

<pre>[progressbar_list id="progressbar_3" color="#400090" width="500" value="30"]</pre>

*.Progress Bar Height

<pre>[progressbar_list id="progressbar_4" color="#FE0A9B" width="400" height="30"  value="40"]</pre>

*.Progress Bar Value

<pre>[progressbar_list id="progressbar_5" color="#4CB848" width="400" height="30"  value="50"]</pre>

*. Progress Bar  Title

<pre>[progressbar_list id="progressbar_6" title="HTML"  color="green" value="60" ]</pre>

*. Progress Bar  Title Color

<pre>[progressbar_list id="progressbar_7" title="HTML"  color="#1E8CBE" value="70" titlecolor="green" ]</pre>

*. Progress bar  Title Size

<pre>[progressbar_list id="progressbar_8" title="HTML"  color="#76A039" value="80" titlecolor="green" titlesize="20" ]</pre>

*. Progress bar Percent Color

<pre>[progressbar_list id="progressbar_four" title="PHP" titlecolor="green" titlesize="20"  color="yellow" percent_color="green" value="90"]</pre>

*. Progress bar Background

<pre>[progressbar_list id="progressbar_10" title="HTML"  color="#00A1DC" value="100" titlecolor="#00A1DC" titlesize="20" percent_color="#086200" bg="#000" ]</pre>
